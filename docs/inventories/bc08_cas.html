Attributes

cas_id

The cas_id attributes is used in 5 of the 6 CASFRI attribute tables and is only described here. The BC08 inventory has the following variations:

  • Header identifier: “BC08”
    • Note: previously, the header identifier included the inventory_standard_cd [converted from=c(“F”,“V”,“I”), to=c(“4”,“5”,“6”); this was dropped on the assumption, to be confirmed, that all data have been converted to “V” type.
  • Source file name: “VEG_COMP_LYR_R1”
  • Name of mapsheet: map_id
  • Object identifier: objectid

identification_id

The attribute identification_id is created from the inventory_standard_cd:

  • mapvalues(inventory_standard_cd, c(“F”,“V”,“I”,“L”), c(4,5,6,0))
  • the value “L” is not defined in CASFRI and has been assigned a value of 0 for now
friList = c("F","V","I","L")
casList = c(4,5,6,0)
x = mutate(x, identification_id = mapvalues(inventory_standard_cd, friList, casList))
